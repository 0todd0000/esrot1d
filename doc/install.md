# Installation



- Install all dependencies listed in [requirements.txt](https://github.com/0todd0000/esrot1d/blob/main/requirements.txt)  , for example:

  - ```
    pip install -r requirements.txt
    ```

- Clone the repository or select the green "Code" button from the [repository home page](https://github.com/0todd0000/esrot1d) to manually download the code

- Add the `./src` folder to the Python path

- Execute `import esrot1d`
